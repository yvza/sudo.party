4.  **Setup Antigravity Manager**
    Akun yang berhasil kita set tadi tinggal kita pakai melalui antigravity manager. Masuk ke menu **Accounts > Add Account > Start OAuth**, lalu login ke gmail yang payment profile nya uda di set ke US tadi.

    Jangan lupa start service di bagian **API Proxy**, nyalakan toggle **Auto Start with App** (biar auto nyala kalau kalian open Antigravity Managernya).

    Set ini di `.zshrc` or `.bashrc` or yang lain, biar ga perlu export-export lagi ketika buka terminal:

    ```bash
    export ANTHROPIC_API_KEY="sk-antigravity"
    export ANTHROPIC_BASE_URL="[http://127.0.0.1:8045](http://127.0.0.1:8045)"
    ```

    Tinggal panggil `claude` diterminal, **Done!**
